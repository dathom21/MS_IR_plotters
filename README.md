Posted here are a collection of Mathematica notebooks for visualizing data, mostly from mass spectrometry or IR spectroscopy experiments. 
The FTIR spectra plotter accepts .dat files and assumes that files are giving transmittance data. If you want to generate the appropriate file from the ascii files generated by PerkinElmer IR spectrometers, you can use the convert_IR_ascii_to_dat notebook file.
The IR spectrum plotter plots intensity vs photon energy, assuming a positive value for intensity.
The Import_CSV notebook will generate a generic plot of CSV data, assuming the first column is x data and subsequent columns are y data.
The IR peak picker will identify maxima in an IR spectrum. It is designed for a positive intensity.
The isotope calculator is designed to calculate exact mass for a given chemical formula. Right now, it is calculating water adducts in a mass spectrum but could be adapted as needed.
The MS spectra plotter needs a .dat file.
The NMR spectra plotter assumes a .CSV file.
The SeeMS extract plotter can be used to plot mass spectra extracted from a LC-MS file using the See-MS software package.
